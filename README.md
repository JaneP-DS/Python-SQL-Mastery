# 🐍 Python-SQL-Mastery
> 파이썬과 SQL의 핵심 개념을 완벽하게 복습하고 기록하는 저장소입니다.

## 🎯 학습 목표
1. **Python**: 기본 문법부터 심화 개념까지 Pythonic한 코드 작성 능력을 함양합니다.
2. **SQL**: 효율적인 데이터 추출을 위한 쿼리 작성 능력을 마스터합니다.
3. **Consistency**: 매일 파이썬과 SQL을 번갈아 학습하며 꾸준한 성장을 기록합니다.

## 📅 학습 방식
- **Python Day**: 핵심 문법 정리 및 실습 코드 작성
- **SQL Day**: 데이터베이스 이론 정리 및 쿼리 문제 풀이

## 📂 저장소 구조
- `/Python`: 변수, 자료형, 제어문, 함수, 클래스 등 파이썬 학습 내용
- `/SQL`: SELECT, JOIN, 서브쿼리, 윈도우 함수 등 SQL 학습 내용
