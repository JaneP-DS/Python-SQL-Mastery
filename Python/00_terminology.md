# 📚파이썬 용어 정리
## 1. 식별자(Identifier)
프로그래밍 언어에서 이름을 붙일 때 사용하는 단어입니다. 주로 변수나 함수 이름 등으로 사용됩니다.
### 🚫식별자 규칙
• **키워드 사용 불가**: 파이썬 예약어(if, for, while 등)는 사용할 수 없습니다. 

• **특수 문자**: 언더바(_)만 허용됩니다.

• **숫자**: 숫자로 시작할 수 없습니다.

• **공백**: 공백을 포함할 수 없습니다.

### 🐍 스네이크 케이스 vs 🐫 캐멀 케이스
파이썬에서는 주로 **스네이크 케이스**를 사용하지만, 용도에 따라 구분합니다.

### 📊 식별자 구분 가이드

| 구분 | 케이스 스타일 | 특징 | 예시 |
| :--- | :--- | :--- | :--- |
| **클래스** | 캐멀 케이스 | 대문자로 시작 | `MyClass` |
| **함수** | 스네이크 케이스 | 소문자 + 뒤에 `()` 있음 | `my_function()` |
| **변수** | 스네이크 케이스 | 소문자 + 뒤에 `()` 없음 | `my_variable` |

## 2. 자료(Literal)
• **자료(Literal)**: 값 자체를 의미합니다. (예: 10, "Hello", True)

## 3. 함수(Function)
•**함수**: 이름 뒤에 괄호 () 가 붙어있다면 함수라고 부릅니다.

•**print()**: 데이터를 출력하는 함수입니다. 괄호 안에 아무것도 넣지 않으면 **줄바꿈(빈 줄)** 을 출력합니다.
